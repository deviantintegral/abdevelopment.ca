To see the code or report issues, go to the sandbox project on drupal.org.

1. Start your selection at the first date
2. End your selection at the last event
From the "My Schedule" page on chicago2011.drupal.org, copy the text of everything from the first date header to your last scheduled event.

3. Save the resulting iCal file, and import it into a calendar application such as iCal or Google Calendar.
